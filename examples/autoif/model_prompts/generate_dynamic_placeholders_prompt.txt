Below is a user query and an instruction template that contains dynamic placeholders. Your task is to generate appropriate values for each placeholder based on the context of the query.

Dynamic placeholders are values that should be relevant to the specific query, such as keywords, phrases, or words that make sense in context.

The input is enclosed in the <query></query> and <instruction></instruction> tags.
The placeholders that need values are listed in <placeholders></placeholders>.

Your output must be a JSON object with:
- "instruction": the instruction with all placeholders filled in with the generated values
- "kwargs": an object containing each placeholder name and its generated value

Guidelines:
- For keyword/word placeholders: choose words relevant to the query topic
- For phrase placeholders: create short, natural phrases related to the query
- For forbidden_words: choose words that would be natural to use but should be avoided
- Make values reasonable and achievable in a response

Example input:
<query>Explain the benefits of regular exercise for mental health.</query>
<instruction>Include keywords {{keywords}} at least {{N}} times in your response.</instruction>
<placeholders>keywords, N</placeholders>

Example output:
```json
{{
    "instruction": "Include keywords \"endorphins\", \"stress relief\", \"mood\" at least 2 times in your response.",
    "kwargs": {{
        "keywords": ["endorphins", "stress relief", "mood"],
        "N": 2
    }}
}}
```

Input: 
<query>{query}</query>
<instruction>{instruction}</instruction>
<placeholders>{placeholders}</placeholders>

Output: include only the JSON object enclosed in triple backticks and nothing else.
